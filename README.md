# Problem Solving & Algorithm Practice

This repository is dedicated to practicing problem solving (PS), studying algorithms, and preparing for coding interviews.

## Platforms

Problems are solved from the following platforms:

* Baekjoon (Expired)
* Programmers
* LeetCode

Each platform has its own directory for better organization and easier navigation.

## Languages

Solutions are implemented using:

* Rust
* C++

The choice of language depends on the problem, performance requirements, and learning goals.

## Purpose

* Improve problem-solving skills
* Strengthen understanding of data structures and algorithms
* Prepare for technical interviews and coding tests
* Experiment with different approaches across multiple languages

## Structure

The repository is organized by platform and language:

```
PS/
├── baekjoon/
│   ├── rust/
│   └── cpp/
├── programmers/
│   ├── rust/
│   └── cpp/
├── leetcode/
│   ├── rust/
│   └── cpp/
```

## Notes

* Each solution focuses on clarity and correctness.
* Some problems may be implemented in both languages for comparison and practice.
* This repository is continuously updated as more problems are solved.
